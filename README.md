# AI-Powered E-Commerce Analytics System

## Project Overview

This is an end-to-end E-Commerce Analytics project built using Excel, SQL, Python, and Power BI.

The project analyzes:

* Orders Dataset
* Customers Dataset
* Payments Dataset
* Products Dataset
